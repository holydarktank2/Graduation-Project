# Graduation-Project

安全道路駕駛模擬器

## Introduction

為提升台灣駕駛的行車觀念，將駕駛行為分為五大觀念，使用者需要在模擬器中駕駛車輛，通關五大觀念內的所有關卡，了解並學習正確的駕駛觀念。

## Required

Logitech G29
